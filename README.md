# 🧾 Cadastro de Contribuinte

Projeto desenvolvido em Java com interface gráfica utilizando Swing, com o objetivo de simular um sistema simples de entrada de dados de contribuintes (Pessoa Física e Jurídica).

---

## 📌 Funcionalidades

* Inserção de dados do contribuinte:

  * Nome
  * CPF
  * CNPJ
  * Cartão
* Seleção do tipo de contribuinte:

  * Pessoa Física
  * Pessoa Jurídica
* Leitura e exibição dos dados no console

---

## 🛠️ Tecnologias utilizadas

* Java
* Swing (Interface gráfica)
* Eclipse IDE

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/JP-GhOsT/prototipo-receita-federal.git
```

2. Abra o projeto no Eclipse

3. Execute a classe principal:

```
Informa_dados.java
```
